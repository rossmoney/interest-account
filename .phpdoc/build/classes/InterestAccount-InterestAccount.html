InterestAccount extends Account

implements InterestAccountInterface

InterestAccount.php : 13

Implements the interest account functionality, extends the basic account with methods to set the users income, set the date interest was last calculated for testing purposes, and calculate the interest. openAccount function is extended to check we have fetched the income as an account must be opened with the income value and cannot be set later.
